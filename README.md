# get-down

A simple to use downloadBinaries tool.

## Get started

### 1. Download binaries

```shell
pnpm download-binaries
```

### 2. Install dependencies

```shell
pnpm install
```

### 3. Build the app

```shell
pnpm build
```

### 2. Run the application

Run the `get-down.exe` file under `./src-tauri/target/release`.
